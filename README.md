## Face-Recognition-and-Matching

**dataset**: https://drive.google.com/drive/u/0/folders/18353vW5EhKf18rFYOMcSOgPM7aZJ2FMv (5 Celebrity Faces Dataset and some images uploaded for matching with the 5 celebrities)
